# dotConfigs

Collection of dot configs for nvim, tmux, and other linux tools for self-use.

This repo use stow to manage dotfiles. see: [Managing Dotfiles with GNU stow](https://news.ycombinator.com/item?id=27137172)

## deps

* ripgrep
* fzf
* ctags
* stow
